PHP модуль Testing
====================

Модуль для организации тестов.

Для подключения модуля достаточно строки:
<?php require_once "./src/Testing.php"; ?>

ПРИМЕЧАНИЕ:
- Верстка и элементы дизайна не входят в модуль.

В каталлоге tests модульные тесты UnitTest.
